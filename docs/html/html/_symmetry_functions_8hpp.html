Radial symmetry function g2 suitable for describing the radial environment of atom i.

g2 is the sum of Gaussians multiplied by cutoff functions. The shifted \g2 is suitable to describe a spherical shell around the reference atom.

Parameters
etaDefines the width of the Gaussians
RsShifts the center of the Gaussians to a certain radial distance
rThe distance between atoms i and j
rcutThe cutoff radius
phiRadial symmetry function g2 value

Radial symmetry function g3 suitable for describing the radial environment of atom i.

g3 is a damped cosine function with a period length adjusted by parameter kappa

Parameters
kappaAdjusting the period length of the damped cosine function
rThe distance between atoms i and j
rcutThe cutoff radius
phiRadial symmetry function g3 value
Note
Care must be taken when using the g3 function. Neighboring atoms can cancel each other’s due to the existence of positive and negative function values. It is recommended to use g3 in combination with other symmetry functions.
